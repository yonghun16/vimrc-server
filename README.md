# vimrc-server

## 1. Install
```bash
git clone https://github.com/yonghun16/vimrc-server ~/vimrc-server
cd ~/vimrc-server
cp .vimrc ~
```

## 2. Update
Run `:PlugInstall`

## 3. Plugins info
- preservim/nerdtree        (파일 탐색기)
- vim-syntastic/syntastic   (문법 검사)
- jiangmiao/auto-pairs      (괄호 자동 완성)
- farmergreg/vim-lastplace  (마지막 커서 위치 기억)
- itchyny/vim-cursorword    (커서 하이라이트)
- tpope/vim-commentary      (주석 토글)
